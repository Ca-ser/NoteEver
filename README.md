# Markdown Editor

一个基于 TypeScript 开发的 Markdown 编辑器，支持实时预览、语法高亮和git实时同步，皆在提供实时快捷的 Markdown 编写体验。

## 项目简介

NoteEver 是一个轻量级 Web Markdown 编辑器，使用 TypeScript 构建，支持 Markdown 实时渲染和常用编辑功能，适用于博客写作、文档编辑等场景。

## 技术栈

- TypeScript
- HTML5
- CSS3
- Markdown Parser
- Electron
- Webpack / Vite

## 功能特性

- 实时 Markdown 预览
- Markdown 语法高亮
- 支持git 一键同步数据
- 支持代码块、图片、链接等语法
- 编辑区与预览区同步滚动
